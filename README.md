# Online-Voting-System
The Voting System project is a simple console-based application developed using Python and MySQL. It is designed to manage the voting process in a secure and organized manner. The system allows users to register with a unique username and password, log in using their credentials, and cast a vote for their preferred candidate.
A key feature of this system is that it ensures “one user, one vote”, preventing duplicate voting by tracking whether a user has already voted. All data, including user details and vote counts, is stored in a MySQL database, which helps maintain accuracy and consistency.

The system also provides functionalities to display the total votes received by each candidate and automatically determine the winner based on the highest number of votes. This makes the voting process transparent and efficient.

Overall, this project demonstrates how Python programming and database integration can be used to build a basic but effective electronic voting system.
